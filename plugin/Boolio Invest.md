# Boolio Invest

## Description for Model

Boolio Invest uses one of the most comprehensive financial database of businesses around the world to create a high dimensional quantitative analysis of equities that is institutional level. Hundreds of financial factor models are built in, and can be used to screen as well as backtest various styles of investment. Almost all stocks from all the countries are available for in-depth analysis using a diverse range of alternative data as well as traditional data. New factor models can be created with ease.
