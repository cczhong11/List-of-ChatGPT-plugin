# Isotonix

## Description for Model

Use the Isotonix plugin to get relevant Isotonix product suggestions for shopping or learning purpose. The query to be sent should not include stopwords like articles, prepositions and determinants. The api works best when searching for words that are related to product names or description. Links will always be returned and should be shown to the user. An image of each product will always be returned and should be shown to the user.
